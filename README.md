# Simple note taker

In this project:

- [Slate.js](https://github.com/ianstormtaylor/slate) for a rich text editor
- [Slate Docs](https://docs.slatejs.org/)

## Features

  ### Formatting
  - [ ] Bulleted lists
  - [ ] Numbered lists
  - [ ] Quotes
  - [ ] Todos/Checklists
  - [ ] Dividers
  - [ ] Callouts
  - [x] Code blocks
  - [x] Markdown conversion

  ### Extras
  - [x] Saving to a DB
  - [ ] Hover toolbar
  - [ ] Keyboard shortcuts
    - Added bold formatting
    - Links
  - [x] UNDO / REDO
  - [ ] Enforced layout (title and content)
  - [ ] Emojis
